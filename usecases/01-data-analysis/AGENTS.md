When analyzing data in this folder:
- Reference specific personas from `@PERSONAS.md` when identifying pain points.
- If analyzing CSV files, suggest Python/Pandas code snippets for calculations if the file is too large for direct context.
- Format analysis reports with:
    1. Executive Summary
    2. Key Findings (with quotes/data evidence)
    3. Recommendations ranked by impact/effort.
- Preferred frameworks:
    - Value Proposition Canvas (Pains, Gains, Jobs)
        - Keep direct quotes for each interview
- Create each artifact as a separate file in the `./artifacts/` folder.
