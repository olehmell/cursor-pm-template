# Kyiv Prioritization Framework

The Kyiv framework is an intuitive and realistic approach to prioritization, offering more transparent communication and budget awareness compared to traditional methods like MoSCoW.

## Categories

- **K**eep for sure: Non-negotiable, essential features that must be included (Minimum Viable Product).
- **Y**es, if we possibly can: High-value features the team is strongly committed to implementing.
- **I**f we have extra budget, we will: Desirable features contingent on personnel availability and resources.
- **V**ery unlikely: Features acknowledged but realistically unlikely to be implemented in the current scope.

## Why Kyiv?

1. **Honest Communication**: "Keep for sure" is a stronger commitment than "Must have", and "Very unlikely" is more honest than "Won't have".
2. **Budget Awareness**: Explicitly acknowledges resource constraints with the "If we have extra budget" category.
3. **Nuanced Phrasing**: Better differentiation between high-value and desirable features.
4. **Psychological Benefits**: Positive language ("Yes, if we possibly can") creates a more collaborative atmosphere.
