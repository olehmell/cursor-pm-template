# PRD Review Prompt

When reviewing a PRD for PetCare:
1. **Check Against Personas:** Does this solve a real problem for Alex, Chloe, or Jordan as described in @PERSONAS.md?
2. **Strategy Fit:** Does this move the needle on our Q1-Q2 2026 Focus (Activation, Notifications, Order Food validation) as per @COMPANY.md?
3. **Product Philosophy:** Does it follow "Low-effort setup" and "At a glance clarity" principles? Is it "Pet-first"?
4. **Competitive Edge:** How does this help us win against Animal ID, Bubibo, or 11pets according to @COMPETITIVE.md?
5. **Technical Feasibility:** Are there obvious technical gaps or performance concerns for React Native/Node.js stack?

Format your output as a collaborative review comment.
